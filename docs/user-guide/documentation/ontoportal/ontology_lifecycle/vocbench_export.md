## Export a semantic artefact from VocBench

Exporting a semantic artefact from [VocBench](https://vocbench.uniroma2.it/) to an OntoPortal installation is a portal-specific feature, available on portals that integrate VocBench — for example **EcoPortal**.

> ℹ️ This feature is portal-specific. Use the portal tabs at the bottom of this page to switch to a portal that provides VocBench (e.g., EcoPortal) and read the corresponding documentation.
