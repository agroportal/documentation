## Create and edit semantic artefacts

[VocBench](https://vocbench.uniroma2.it/) is a collaborative, web-based platform for creating, editing and publishing semantic artefacts. It is integrated by some OntoPortal installations of the Alliance — for example **EcoPortal**.

> ℹ️ This feature is portal-specific. Use the portal tabs at the bottom of this page to switch to a portal that provides VocBench (e.g., EcoPortal) and read the corresponding documentation.
