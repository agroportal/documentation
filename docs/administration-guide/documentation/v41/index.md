# OntoPortal Administration Guide — Appliance v4.1

Installation and maintenance documentation for OntoPortal Appliance v4.1.
You can find
<a href="{{site.baseurl}}/administration/general/v41">general information</a> about the {{site.opva}}
and its installation process, including descriptions
of the <a href="{{site.baseurl}}/administration/steps/v41">installation steps</a> as well as
<a href="{{site.baseurl}}/administration/ontologies/v41">managing ontologies</a>.

To learn how to use a running {{site.opva}}, see the <a href="{{site.baseurl}}/user_guide"> User Guide </a>,
or to use the API of your own appliance see the documentation generated at (found at `https://{my_appliance_hostname:8080}/documentation`).

The {{site.opva}} is provided by [Stanford BMIR](https://bmir.stanford.edu/) which keeps trace of all the uses and assigns free licenses. 

The documentation focuses on system administration for Appliance v4.1.
